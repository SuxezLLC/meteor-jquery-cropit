# jQuery Cropit

Provides jQuery [Cropit](http://scottcheng.github.io/cropit/) for Meteor.

## Install

    meteor add suxez:jquery-cropit

## Author

 * [Scott Cheng](https://github.com/scottcheng) is the author of the original Cropit library
